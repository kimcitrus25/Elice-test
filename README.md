# 🧩 Elice-test: 바이브 코딩(Vibe Coding) 실습 저장소

인공지능 에이전트와 대화하며 코드를 작성하고, 실제 배포까지 경험해보는 실습 프로젝트입니다. "코드를 짜는 것"이 아니라 "AI와 협업하여 가치를 만드는" 새로운 개발 패러다임을 체험합니다.

## 🚀 실습 목표
- AI 에이전트 정의 및 연결
- AI와의 자연스러운 대화형 코딩 (Invoke)
- 복잡한 작업을 잘게 나누는 스몰 스텝(Small Step) 전략 이해
- 프로젝트의 헌법(CLAUDE.md)을 활용한 컨텍스트 제어
- 자율 디버깅을 통한 에러 복구 프로세스 체험

## 🛠 주요 기능
- **Interactive UI**: 웹 브라우저에서 직접 AI와 대화하며 코드를 실행해볼 수 있는 대화형 실습 환경
- **Real-time Terminal**: 실시간으로 생성되는 AI의 응답과 터미널 출력을 확인
- **Vibe Coding Flow**: 프롬프트 엔지니어링을 통한 효율적인 개발 흐름 체득

## 💡 학습 단계
1. **AI 정의**: `ChatOpenAI` 객체 생성 및 엔드포인트 연결
2. **첫 대화**: `invoke`를 통한 AI 에이전트와의 첫 소통
3. **작업 분할**: 빅 스텝을 스몰 스텝으로 나누어 효율 극대화
4. **컨텍스트 설계**: `CLAUDE.md`로 프로젝트 규칙 학습시키기
5. **자율 디버깅**: 코드 에러를 스스로 분석하고 복구하는 에이전트 경험
